Прогоамма реализует гистерезисное управление нагревателем и вентилятором для поддержания температуры в заданном диапазоне без частых переключений (защита от дребезга реле).

Как работает:

· Нагреватель включается при ≤21°C и выключается при ≥25°C. Между 21–25°C состояние не меняется (зона нечувствительности).
· Вентилятор включается при ≥35°C и выключается при ≤31°C. Между 31–35°C — бездействие.

Переменные:

· temp — текущая температура (ручной ввод для теста).
· heater / hold — состояния нагревателя и вентилятора.
· heatTemp_On/Off и holdTemp_On/Off — пороги срабатывания.

Подходит для теплиц, инкубаторов или любых систем терморегуляции с релейными выходами.
